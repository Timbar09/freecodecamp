# Build a Bookstore Page

## Step 1

Creating the HTML boilerplate. Should include the `<!DOCTYPE html>` declaration and `html` and `head` elements with a lang attribute to the html element and set it to "en".

## Step 2

Add the `title` element inside the `head` element. Set the page title to `XYZ Bookstore Page`.

## Step 3

Inside the `head` element, add the `<meta charset="UTF-8">` element and add a `body` element below the head section. This is where all of your visible page content will go.

## Step 4

In this step add an `h1` element with the text `XYZ Bookstore`.

## Step 5

Below the `h1` element, add a `p` element with this text: `Browse our collection of amazing books!`.

## Step 6

Below the `p` element, add a `div` element. This div will be a container for your cards.

## Step 7

The `class` attribute is used to identify one or more elements for styling.

Add a class attribute to your `div` element and set its value to `card-container`.

## Step 8

Inside the element having a class of `card-container`, create another `div` element. This div will represent the first card.

Add a `class` attribute to this `new div` element and set the value of the class attribute to `card`.

## Step 9

The `id` attribute adds a unique identifier to an HTML element.

`id` values cannot contain spaces and should only contain letters, digits, underscores, and dashes.

Add an `id` attribute to your element having a class of card and set its value to `sally-adventure-book`.

## Step 10

Inside the first element having a class of `card`, add an `h2` element with the text `Sally's SciFi Adventure`.

## Step 11

Below the `h2` element in the first element having a class of `card`, add a `p` element with the following text: `This is an epic story of Sally and her dog Rex as they navigate through other worlds`.

## Step 12

The `button` element is used to create clickable buttons on a webpage. Buttons are interactive elements that users can click to perform actions.

Add a `button` element inside the element that has a class of `card`, give the button a `class` attribute set to `btn`, and the text `Buy Now`.

## Step 13

Now create a `second` card. Add another `div` element with the `class` attribute set to `card`. Notice how you can reuse the same class name for multiple elements to apply consistent styling.

Add an `id` and set its value to `dave-cooking-book`. Remember that each id must be unique.

Inside the element, add an `h2` element with the text `Dave's Cooking Adventure`, add a `p` element with this text:

`This is the story of Dave as he learns to cook everything from pancakes to pasta, one recipe at a time`.

Add a `button` element with the `class` attribute set to `btn` and the text `Buy Now`.

Both button elements now share the same class, which means they can be styled consistently together.

## Step 14

Below the `card-container` element, add a new `p` element with this text:

`Review your selections and continue to checkout.`

Below the `p` element, create a `div` element with the `class` attribute set to `btn-container`. This container will group your navigation button elements.

## Step 15

Inside the element with a class of btn-container, add two `button` elements:

First button:

`Id`: view-cart-btn, `Class`: btn, `Text`: View Cart

Second button:

`Id`: checkout-btn, `Class`: btn, `Text`: Checkout

Congratulations! You have successfully built the structure of a bookstore page using divs, classes, and ids to organize your content.
